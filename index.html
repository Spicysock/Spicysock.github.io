How to make a website

This is a guide on how to create a website

Step 1 - Use a text editor

Any Text editor will work

For Example - Notepad

I used Notepad++ to make this website

This page in Notepad++

This page in Notepad

Step 2 - Create the file

A website uses .html files

An html file starts with the code <!DOCTYPE html> and <html>

The visible content of the page is contained in <body>

<title> specifies a title for a page

<p> is used for paragraphs

All code is contained within brackets <>, and ends with the same code, but with a forward slash </>

The first lines of code used for this page

Step 3 - Add Content

Add whatever you want to add to the page

To add images, use <img src='image url'>

To add links, use <a href="url"> Link Text

Step 4 - Change Page Style

The size, font, and color of text can be changed

The color can be changed by using <p style="color:(color)>

The font size can be changed by using <p style="font-Size:#px;">

The color of the background can be changed using <p style='background-color:(color);">

The style information for the page, can also be stored in a seperate style sheet file

Step 5 - Host the Website

The website needs to be hosted, or it is just a file

The website can be hosted by any hosting service, both free and paid
